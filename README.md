## 📌 About

A sleek, modern, fully responsive **personal portfolio website** built from scratch using pure HTML, CSS, and JavaScript — no frameworks, no bloat. Features a cinematic loading screen, animated hero section, dark/light theme toggle, project filtering, and a smooth single-page experience.

---

## 🌟 Features

- ⚡ **Animated Loading Screen** — Custom SVG chip with flowing circuit traces
- 🎨 **Dark / Light Theme** — Persisted via `localStorage`, respects system preference
- ✍️ **Typewriter Effect** — Cycles through your roles automatically
- 🌠 **Particle Hero** — Dynamic floating particles in the hero section
- 🖱️ **Cursor Glow** — Smooth radial glow that follows the mouse
- 📱 **Fully Responsive** — Mobile-first, hamburger menu on small screens
- 🗂️ **Project Filter** — Filter projects by category (Web / Mobile / AI-ML)
- 🎞️ **Scroll Animations** — Elements fade in via IntersectionObserver
- 🔒 **SEO Ready** — Meta tags, Open Graph, and semantic HTML throughout

## 🚀 Getting Started (Local Dev)

No build tools required — just open the file in your browser.

```bash
# Clone the repository
git clone https://github.com/crz02/Portfolio-Irfan.git

# Navigate into the folder
cd Portfolio-Irfan

# Open directly in browser (macOS)
open index.html

# Or use a live-server (if you have it installed)
npx live-server .
```

## 🌐 Deployment

This site is deployed via **GitHub Pages** from the `main` branch.

| Setting | Value |
|---------|-------|
| Branch | `main` |
| Root | `/` (root) |
| URL | [https://crz02.github.io/Portfolio-Irfan/](https://crz02.github.io/Portfolio-Irfan/) |

To deploy your own fork:
1. Fork this repo
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, `/ (root)`
4. Click **Save** — your site will be live in ~1 minute
